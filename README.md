## Setup Instruction

1) Clone the repository  
2) Run `npm install` (it will install all the packages from the `package.json`)  
3) Create a database in PostgreSQL using pgAdmin or psql CLI  
4) Rename `.env.sample` to `.env` and add all of your environment variable values  
5) Download & install ffmpeg on your system from here: [https://www.ffmpeg.org/download.html](https://www.ffmpeg.org/download.html) 
6) Download and install redis
7) Run the project by executing the `npm run dev` command
